# 45. Jump Game II

## Problem
Find the minimum number of jumps to reach the last index.

---

## Approach
Greedy:

- Track current range (currentEnd)
- Track farthest reachable index
- When reaching currentEnd → increase jumps

---

## Complexity
Time: O(n)
Space: O(1)

---

## Pattern
Greedy (Range Expansion)
